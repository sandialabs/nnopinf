NN-OpInf
============= 
NN-OpInf is a PyTorch based approach to operator inference that utilizes composable, structure preserving neural networks to represent nonlinar operators. Operator infernce, as made popular by `Peherstorfer, Willcox, and co-authors <https://willcox-research-group.github.io/rom-operator-inference-Python3/source/opinf/intro.html>`_, is an approach for inferring low-dimensional systems from data. Classically, OpInf infers polynomial models for the system dynamics. Numerous systems of intereset, however, do not admit such polynomial structure. NN-OpInf addresses this challenge by parameterizing operators with neural networks.
